Other Projects / Code Examples

Viginere

A small encryption algorithm written in C that takes two input strings. Each character of the first string is assigned an integer value and is then used as a cypher to encode each corresponding character in the second string.

Speller

A program written in C that uses a hash algorithm to store the strings from dictionary file. This has the effect of making retrieval of these strings more efficient when used to spell check a documnent.

Filter

A program that accepts a bitmap image as an input. The code loops through the rgb attributes of each pixel and is able to output the file with various filters such as sepia, reverse, grayscale and blur.

DNA

A Python program that takes a text file containing a long string of DNA sequences and a CSV of candidates and their DNA profiles . The program searches through this sequence looking for specific patterns and the number of times these patterns repeat for each sequence. From this information the program can ascertain who the DNA sequence belongs to from the candidate CSV file.

World Cup

A Python program that takes a CSV as input conatining footbal teams, their rating and the order in which they will play the first round. The program then simulates the tournament and produces ratings for each team's probability of winning the World Cup.
